# Benchmark corpus

Labeled PR descriptions for offline evaluation. **Not required at runtime** for the API or frontend.

| File | Purpose |
|------|---------|
| `quality_prs.jsonl` / `slop_prs.jsonl` | Labeled examples |
| `evaluate.py` | Full benchmark (optional sentence-transformer model) |
| `_benchmark_runner.py` | Fast benchmark (`make benchmark`) |
| `adversarial_test.py` | Anti-gaming scenarios (`make adversarial`) |
| `generate_corpus.py` | Regenerate JSONL (`make corpus`) |
| `synthetic_expansion.py` | Extra synthetic PR templates for corpus generation |

Results JSON files in this folder are regenerated by the scripts above.
